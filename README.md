Project Overview:
CareConnect is a web-based health platform that helps users monitor symptoms, track medications, and log their daily mood. The system uses AI to provide basic health insights and promote better awareness of physical and mental well-being.

Features:
- AI-powered symptom checker
- Medication prescribing
- mental health advisor

Technology:
React.js, Tailwind CSS, Node.js, Express.js, OpenAI API

Goal:
To create a simple digital platform that encourages preventive healthcare and helps users manage their daily health more effectively.
